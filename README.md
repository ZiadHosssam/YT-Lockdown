# YT Lockdown

YT Lockdown is a Chrome extension that helps you focus on one YouTube video at a time.

When lockdown is active:
- The current video tab becomes distraction-free.
- Other YouTube tabs are blocked until the locked video ends or you force unlock.
- YouTube navigation away from the active video is restricted.

## Features

- Lock into one video until it ends
- Hide recommendations, comments, feed, and extra distractions
- Cross-tab lock: blocks YouTube usage in other tabs while active
- Force unlock option from popup

## Permissions

- `storage`: Save lock state.
- `tabs`: Communicate with the active tab from popup.
- `activeTab`: Interact with the currently active tab.
- `*://*.youtube.com/*`: Apply lockdown logic only on YouTube pages.
