# Privacy Policy - YT Lockdown

Effective date: 2026-03-25

YT Lockdown does not collect, sell, or share personal data.

## Data usage

The extension stores only local settings in your browser using extension storage:
- Current lockdown state
- Current locked video identifier
- Internal lock session metadata used to enforce lock behavior across tabs

This data:
- Stays on your device
- Is not transmitted to external servers by the extension
- Is used only to provide extension functionality

## Network access

YT Lockdown runs on YouTube pages (`*://*.youtube.com/*`) and does not send user data to third-party services.
